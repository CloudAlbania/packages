CloudAlbania Package Repository

Welcome to the CloudAlbania package repository. This repository provides APT (Debian/Ubuntu) and YUM (RHEL/CentOS/AlmaLinux) packages for internal and public use. The list of available packages to install is at https://repo.cloudalbania.com/

Use the instructions below to configure your system and install packages.

APT (Debian / Ubuntu)

Add dependencies:

apt install -y sudo ca-certificates

Add the repository:

echo "deb [trusted=yes] https://repo.cloudalbania.com/apt/ /" | sudo tee /etc/apt/sources.list.d/cloudalbania.list

Update package list:

sudo apt update

Install a package:

sudo apt install <package-name>

YUM / DNF (RHEL / CentOS / AlmaLinux)

Install dependencies:

dnf install sudo -y

Create a repo file:

sudo tee /etc/yum.repos.d/cloudalbania.repo <<FILE
[cloudalbania]
name=CloudAlbania Repository
baseurl=https://repo.cloudalbania.com/yum
enabled=1
gpgcheck=0
FILE

Install a package:

sudo yum install <package-name>
Or with dnf: 
sudo dnf install <package-name>

Notes

  • Replace <package-name> with the actual package name.
  • Make sure your system has network access to repo.cloudalbania.com.
  • If you encounter issues, contact your system administrator.
